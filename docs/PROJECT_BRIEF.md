# Project Brief

## What This Is

Heartspace is a privacy-first relationship wingman and memory tool by Faith
Cheltenham.

It supports adult users with message practice, date prep, relationship memory,
social confidence, after-date reflection, and reality checks. The product is
designed to protect consent, honesty, reciprocity, safety, and real-world
connection.

## What It Is Not

Heartspace is not a dating app, matching service, swiping platform, message
automation bot, scraper, impersonation tool, catfishing tool, or manipulation
engine.

## Why It Exists

Relationship tools often reward speed, performance, and dependence. Heartspace
is built around a different premise: people deserve tools that help them think,
remember, practice, and choose with dignity.

## Public Repository Purpose

This repository presents the project publicly while protecting the private
engine, source code, workflows, prompts, account data, deployment systems, and
operational infrastructure.
