# Security

## Public Repository Boundary

Do not open issues or pull requests containing:

- API keys, tokens, passwords, cookies, private keys, OAuth files, or recovery
  phrases;
- private account data or exports;
- deployment host details;
- database contents;
- medical, legal, benefits, family, or admin records;
- unpublished creative work not already included in this repository.

## Reporting

Report public-surface security concerns through Faith Cheltenham's normal
contact channels at https://faithcheltenham.com/.

Do not attempt to access private systems, production routes, databases,
dashboards, admin panels, or authentication flows.
