# Heartspace

## Hero

Heartspace is a privacy-first relationship wingman and memory system by Faith
Cheltenham.

It helps adults practice messages, prepare for dates, remember relationship
context, check assumptions, and protect their own pacing without turning
connection into swiping, scraping, impersonation, or manipulation.

Recommended featured image: `assets/hero/hero-image.png`

## What This Project Is

Heartspace is a protected relationship-support tool. It is built around message
practice, date prep, memory keeping, reality checks, and privacy controls.

## Why It Matters

People deserve tools that help them think clearly and connect honestly.
Heartspace treats relationships as real human contexts, not funnels, targets,
or engagement loops.

## How It Works

Users bring private relationship context into a protected workflow. Heartspace
helps organize the next step through coaching modes such as Message Lab, Date
Prep, Relationship Memory, and Reality Check. The output stays user-owned.
