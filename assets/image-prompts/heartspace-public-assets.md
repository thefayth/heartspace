Use only if the existing hero asset is replaced or expanded after Faith review.

## Hero

Create a warm premium hero image for Heartspace, a privacy-first relationship
wingman and memory app. Show emotional safety, private reflection, consent,
and real-world connection through abstract visual cues. No readable text, no
logos, no private screens, no dating-app swiping motifs.

## Social Card

Create a public-safe social card background for Heartspace with plum, rose,
cream, and teal tones. The image should feel human, thoughtful, and protective,
not generic SaaS and not explicit romance advertising.

## Workflow Illustration

Create a high-level workflow illustration showing a user, private reflection,
ethical review, user-owned output, and public documentation separated from
protected systems.
